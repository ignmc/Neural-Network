El programa se ejecuta desde el archivo main.py. En él también se pueden ajustar las variables usadas, forma de la red, learning rate y epochs.
Se recomienda utilizar un ambiente virtual. En él se pueden instalar las dependencias con el comando 'pip install -r requirements.py'
Una vez que las dependencias están instaladas se puede ejecutar el programa con el comando 'python main.py'

Las dependencias utilizadas por el programa son pandas y matplotlib, y por supuesto también sus dependencias respectivas

Probado con python 3.6
